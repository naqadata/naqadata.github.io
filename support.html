Naqafin Support

Last updated: June 16, 2026

Naqafin is an independent Roku client for Jellyfin-compatible media servers. If you need help with the Naqafin Roku app, please use the support options below.

Contact Support

The preferred support channel is the Naqafin Roku issue tracker: https://github.com/naqadata/naqafin-roku/issues.

When reporting an issue, please include your Roku model, Roku OS version, Naqafin app version, Jellyfin server version, and a short description of what happened.

Do Not Share Sensitive Information

Please do not include passwords, authentication tokens, private server URLs, personal media files, or other sensitive information in public support requests.

Server And Media Issues

Naqafin does not provide media content, Jellyfin server hosting, subscriptions, or user accounts outside your own Jellyfin-compatible server. Problems with your server configuration, media files, network, plugins, or content rights may need to be resolved on the server you control.
